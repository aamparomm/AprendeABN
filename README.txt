## Descripción del proyecto

En este proyecto se incluye todo el código implementado a lo largo de la elaboración del 
Trabajo de fin de grado denominado "Dando soporte a la metodología ABN de aprendizaje de matemáticas en etapas tempranas".
El objetivo de este proyecto es implementar varios ejercicios de infantil y primaria
que permitan practicar la metodología ABN.
